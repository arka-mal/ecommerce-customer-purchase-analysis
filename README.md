# E-Commerce Customer Purchase Analysis

Data Fundamentals Capstone Project

---

## Overview
This project performs an end-to-end analysis of e-commerce customer purchase data using **SQL, Python (Pandas), and Excel**.  
It extracts actionable insights to support data-driven business decisions.

---

## Problem Statement
Although transaction data is collected (customer, product, city, price, quantity), it is not effectively analyzed.  
This makes it difficult to identify:
- Top spending customers
- Most popular products
- Highest revenue-generating cities

---

## Solution
A multi-tool analysis pipeline was built using:
- **SQL** → Data querying & aggregation  
- **Excel** → Interactive dashboards & summaries  
- **Python (Pandas + Matplotlib)** → Automated analysis & visualization  

---

## How to Run
```bash
pip install pandas matplotlib
python analysis.py
```

---

## Key Insights
- 🏆 Mumbai is the top revenue-generating city  
- 💻 Laptops generate the highest revenue  
- 🎧 Headphones & Mobiles are the most sold products
